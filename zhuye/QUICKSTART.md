# 快速开始指南

## 🚀 一键安装

### Windows 用户
双击运行 `install.bat`

### Linux/Mac 用户
```bash
chmod +x install.sh
./install.sh
```

## 📝 手动配置步骤

### 1. 填写服务器密码

打开 `.env` 文件，填入实际密码：

```env
ROUTER_PASSWORD=<SECRET_7a4a56f8>
NAS_PASSWORD=<SECRET_33167e60>
CLOUD_PASSWORD=<SECRET_33167e60>
OPENCLAW_PASSWORD=<SECRET_33167e60>
```

### 2. 启动服务器

```bash
npm start
```

### 3. 访问主页

打开浏览器访问：http://localhost:3000

## 📋 功能说明

### 左侧侧边栏

1. **时钟**：实时显示当前时间、日期，节假日会特殊显示
2. **日历**：可点击 `<` `>` 切换月份，点击日期显示当天备忘录
3. **备忘录**：
   - 在输入框输入内容，点击"添加"或按回车添加备忘录
   - 点击备忘录右侧的 `×` 删除备忘录
   - 数据保存在浏览器本地，清除数据会丢失
4. **服务器状态**：自动检测各服务器状态，显示在线状态、延迟和负载

### 主内容区

**个人简介卡片**：编辑 `index.html` 中的以下内容自定义：

```html
<h2>你的名字</h2>
<p class="profile-title">职位 / 头衔</p>
<p class="profile-description">个人简介...</p>
```

替换 `avatar.svg` 为自己的头像（推荐使用 SVG 或 PNG 格式）。

## 🐛 常见问题

### Q: 服务器状态显示"检测失败"？

**A:** 检查以下几点：
1. `.env` 文件中的密码是否正确
2. 服务器是否开机并联网
3. SSH 服务是否正常运行（默认端口22）
4. 防火墙是否允许SSH连接
5. 网络是否能访问目标服务器

### Q: 端口3000被占用？

**A:** 修改 `.env` 文件中的 `PORT=3000` 为其他端口，如 `PORT=8080`

### Q: 备忘录数据丢失？

**A:** 备忘录存储在浏览器的 localStorage 中，以下情况会丢失：
- 清除浏览器数据
- 使用隐私/无痕模式
- 更换浏览器或设备
建议定期备份重要备忘录。

### Q: 如何添加更多服务器？

**A:** 修改 `server.js` 中的 `serverConfigs` 对象，添加新服务器配置。
