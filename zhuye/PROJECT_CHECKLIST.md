## 🎯 下一步操作

### 立即执行：

1. **安装依赖**
   ```bash
   npm install
   ```

2. **填写密码**
   编辑 `.env` 文件，填入所有服务器密码

3. **启动服务器**
   ```bash
   npm start
   ```

4. **访问主页**
   浏览器打开 http://localhost:3000

### 自定义：

1. **修改个人简介**
   编辑 `index.html` 中的个人信息

2. **更换头像**
   替换 `avatar.svg` 为自己的头像

3. **调整主题色**
   修改 `style.css` 中的渐变色

4. **添加更多服务器**
   修改 `server.js` 中的 `serverConfigs`

## ⚠️ 重要提醒

1. **安全第一**：`.env` 文件包含敏感密码，切勿上传到Git
2. **密码配置**：确保 `.env` 中所有密码已正确填写
3. **网络连接**：服务器需开放 SSH 端口（默认22）
4. **浏览器兼容**：建议使用现代浏览器（Chrome/Firefox/Safari）
